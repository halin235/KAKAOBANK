# 별자리 적금 ✨

**단순 저축을 넘어 게이미피케이션과 금융을 결합한 리텐션 강화 모델**입니다. 운세·성좌·보상 루프로 매일 돌아오게 만들고, 그 안에서 적금 행동과 금리 메시지를 자연스럽게 연결합니다.

### Constellation Savings — PRD 2.0 하이퍼-콘텍스트 적금 데모

> 당신의 사주(四柱)가 오늘의 저축 금액과 우대금리 내러티브를 풍부하게 만듭니다.  
> 생년월일로 계산한 **60갑자 일주**에 맞는 금융 상품 큐레이션과 성좌 세레모니를 제공하는 카카오뱅크 스타일 모바일 적금 UI입니다. *(실제 카카오뱅크 서비스와 무관한 포트폴리오 프로젝트입니다.)*

---

## 📌 서비스 개요

**별자리 적금**은 전통 명리학의 일주(日柱) 계산과 현대 금융 상품 추천을 결합한 **하이퍼-콘텍스트 적금** 데모입니다.

- **사주 기반 우대금리·만기 시뮬레이션**: 일주 케이스(A/B/C)에 따라 추천 상품과 재물운 메시지를 매핑하고, 납입 금액·기간을 바탕으로 만기 예상 이자(세전 범위)를 계산합니다.
- **60갑자 큐레이션**: 율리우스 적일 기반 일주 산출 후 60개 일주 각각에 재성·비겁·인성 성향과 맞는 상품 라인을 연결합니다.
- **성좌·룰렛 게이미피케이션**: 행운 금액 룰렛, 파티클 버스트, 가입 완료 후 성좌 점등 애니메이션으로 전환 순간의 몰입과 재방문 동기를 강화합니다.

| 영역 | 내용 |
|------|------|
| **60갑자 큐레이션** | 일주별 케이스 매핑, 개인화 메시지, 바텀시트 상품 상세 |
| **행운 금액 룰렛** | 회전·카운트업·황금 파티클로 저축 금액 확정 경험 |

## ☁️ Vercel 배포 · 환경 변수

- 시크릿·API 키는 **`NEXT_PUBLIC_` 접두사로 노출하지 마세요.** 클라이언트 번들에 포함됩니다.
- OG URL·canonical 등 공개 사이트 주소만 필요하면 Vercel 프로젝트 환경 변수에 설정합니다.

| 변수 | 필수 | 설명 |
|------|------|------|
| `NEXT_PUBLIC_SITE_URL` | 선택 | 배포 도메인 (끝 슬래시 없이), 예: `https://your-app.vercel.app` |

로컬에서는 저장소 루트의 `.env.example`을 참고해 `.env.local`을 만들 수 있습니다.
